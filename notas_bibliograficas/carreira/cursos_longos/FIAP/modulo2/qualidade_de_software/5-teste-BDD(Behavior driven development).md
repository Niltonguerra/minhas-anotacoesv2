# Explicação dos tópicos:

## o que é?
- é uma técnica que encoraja a colaboração entre pessoas desenvolvedoras, setores de qualidade e pessoas não técnicas ou de negócios em um projeto de software.
- basicamente é uma conversa com quem quer o projeto para saber as necessidades dele.


## características:
- usa a linguagem Gherkin
	- se parece com linguagem natural
- se foca no comportamento do sistema da perspectiva do usuário.
