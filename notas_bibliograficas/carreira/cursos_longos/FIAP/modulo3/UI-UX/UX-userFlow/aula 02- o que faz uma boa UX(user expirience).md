## requisitos para uma boa UX: 
![[Pasted image 20250806182236.png]]


### usabilidade
- deve ser fácil e simples de usar
	- navegação clara
	- controles bem posicionados
	- feedback automático
### acessibilidade
- deve ser acessível
	- escolha das cores
	-  tamanho de fonte
### utilidade
- deve realmente ajudar o usuário

### desempenho
- deve ser livre de erros
	- ou pelo menos avisar sobre o problema
- o sistema deve ter disponibilidade

### desejabilidade
- não deve ser feio
- deve ser agradável o uso dele
- o usuário deve se sentir confortável ao usar o aplicativo

### relevância
- deve ser relevante para a sua persona
- deve ter o que mais importa para o usuário no momento
	- por exemplo as falar sobre as taxações do trump para investidores e etc
- (dúvida isso não entra na utilidade?)

### confiança e credibilidade
- deve ser um design profissional o produto

### engajamento
- fazer o usuário sempre voltar no seu produto
